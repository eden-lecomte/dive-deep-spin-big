# Victory GIFs

Drop your victory celebration GIF files into this folder!

## How to use:
1. Add any `.gif` files to this folder
2. The victory modal will automatically pick a random GIF from this folder
3. No code changes needed - just drag and drop!

## Notes:
- Only `.gif` files are detected (case-insensitive)
- GIFs are cached for 1 minute for performance
- If you add new GIFs, refresh the page to see them
- GIFs are displayed randomly each time the victory modal appears
